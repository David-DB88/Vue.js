# my-vue
Application in vuejs 3 where the most important approaches in vue are implemented.

Observable WATCH and computable COMPUTED properties,
Dynamic pagination. Infinite Ribbon. Intersection API,
VUE-ROUTER. Page navigation,
Custom directives V-INTERSACTION and V-FOCUS,
Mixins.
Vuex. Global application state,
Composition API.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
